## **Tenant Features**
1. Wallet / Savings  
   - Deposit money → wallet_balance updated  
   - Verified badge proportional to deposit → badge visible for apartments ≤ deposit  
   - Badge removed on withdrawal or rent/inspection payment  
   - Can inspect apartments ≤ verified deposit limit for free  

2. Multi-apartment Inspections  
   - Inspections above deposit limit → pay inspection fee (5k)  
   - Fee auto-split: 2k landlord, 2k platform, 1k agent if referred  

3. Rent / Shortlet Payments  
   - Rent paid via platform → landlord receives full rent  
   - Platform takes 5% commission  
   - Agent receives 1% of 5% for first rent if landlord referred  

4. Badge & Deposit Logic  
   - Deposit increases badge coverage (higher-budget apartments)  
   - Withdrawal triggers 2% fee, badge removed  
   - Deposit used to pay rent → badge removed  

5. KYC Integration  
   - Required for tenants depositing above threshold (e.g., ₦500k)  
   - Upload ID, proof of address, optional selfie  
   - Admin approves/rejects
